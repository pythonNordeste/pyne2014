O evento

Python Nordeste é o maior evento regional voltado aos desenvolvedores da linguagem de programação Python (open-source). Criada pelos apaixonados por simplicidade, a comunidade Python nordestina tem a missão de cativar novos programadores para a linguagem e fortalecer a região com as tecnologias mais recentes usadas no Brasil e no mundo.

Aproveite o calor da Bahia para aprender algo novo com quem manja do assunto.
